# Slang.D Gaussian Splatting Rasterizer

This repository contains an implementation of the [Slang.D accelerated Gaussian Splatting rasterizer](https://github.com/google/slang-gaussian-rasterization/) that has been modified for [Eulerian Gaussian Splatting with Hashed Probability Pyramids](https://github.com/polanskers/eulerian-splatting).
